The API is designed to utilize a Bearer token with an HTTP authorization header.

```bash
'Authorization': 'Bearer <token>'
```

## API Tokens

You can generate the API key in the SofToPos dashboard after adding your website. You can generate multiple tokens for various devices in the manage terminals section of the website detail view.
