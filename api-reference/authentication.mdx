The API is built for use Bearer token with http authorization header

```bash
'Authorization': 'Bearer <token>'
```

## API Tokens

You can generate the API key in the SofToPos dashboard after adding your website, you can genrate mutiple tokens for mutiple devices in the manage terminal section of website detail view.
