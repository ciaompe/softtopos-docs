---
title: "Get Request"
api: "GET https://api.softtopos.be/api/v1/requests/get/{uuid}"
description: "This method requires a UUID as a URL parameter, which you can find in the Request Create method response. Once you have obtained the UUID, you can send the GET request with the correct UUID to retrieve payment information and its corresponding statuses."
---

### Response

<ResponseField name="total" type="number">
    This represents the payment total, rounded to 2 decimal places.
</ResponseField>

<ResponseField name="commission" type="number">
    This represents the calculated payment commission for the SoftoPos API.
</ResponseField>

<ResponseField name="meta" type="array">
   This should be an array. It returns what has been submitted in the create request.
</ResponseField>

<ResponseField name="success" type="boolean">
    This indicates the payment success status.
</ResponseField>

<ResponseField name="failed" type="boolean">
    This indicates the payment failure status. 
</ResponseField>

<ResponseField name="ref" type="boolean">
    This returns the same data that was submitted in the create request.
</ResponseField>

<ResponseField name="transaction" type="object">
    This contains all the transaction data related to the request, and is only returned if the payment success status is true.
</ResponseField>

<ResponseField name="status" type="number">
  This indicates whether the call was successful or not. A status code of 200 is returned if the call was successful and a status code of 400 is returned if the call was not successful.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://api.softtopos.be/api/v1/requests/get/a2e66a51-0d64-45ce-adb7-c38d51cc200f' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json' \
--data ''
```

</RequestExample>
