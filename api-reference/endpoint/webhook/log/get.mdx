---
title: "Get Logs"
api: "GET https://api.softtopos.be/api/v1/webhook/log"
description: "The webhook log contains all the logs related to the specified website. Each webhook log includes status, response, and payload"
---

### Response

<ResponseField name="status" type="string">
    The status shows the response status of your website webhook endpoint. If your webhook endpoint returns a "404 Not Found" status, it will be recorded as "404" in the status field of the webhook log.
</ResponseField>

<ResponseField name="payload" type="array">
    The payload includes all information related to the request, including the "ref" and "meta" information.
</ResponseField>

<ResponseField name="response" type="string">
    This displays the response from your website's webhook endpoint.
</ResponseField>

<ResponseField name="created_at" type="datetime">
    This displays the date and time when the log was created.
</ResponseField>

<RequestExample>
```bash Example Request
curl --location 'https://api.softtopos.be/api/v1/webhook/log' \
--header 'Accept: application/json'
```
</RequestExample>
